# Notepad

An efficient note management tool designed for simplicity and quick access. Users can create and store notes through a clear interface. Whether jotting down ideas or keeping track of information, everything stays organized in one place.

## Instructions for Build and Use

[Software Demo](https://youtu.be/k8a5oyid_bA)

Steps to build and/or run the software:

1. Open the project folder in Visual Studio Code.
2. Open a terminal and run npm install to install the project dependencies.
3. Run npm run build and npm run build:frontend to compile the TypeScript files.
4. Run npm start to start the application.
5. Open your browser and go to the local host address shown in the terminal.

Instructions for using the software:

1. Open the application in your browser.
2. View existing notes displayed on the page.
3. Select a note from the left panel to view.
4. Create, edit, and delete notes using the icon buttons.
